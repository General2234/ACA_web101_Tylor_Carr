What does doctype do at the top of your code?

Doctype specifies the rules for the markup language so browsers render the content correctly. We should always add Doctype declaration to our HTML documents so the browser knows what type of document to expect (Sends instruction to the web browser about what version of the markup language the page is written in).

Explain How A Browser Determines What Elements Match a CSS Selector

Browsers filter out elements in the DOM according to the key selector, and traverse up its parent elemnets to determine matches, the shorter the length of the selector chain, the faster the browser can determine if element matches selector (Selector p span- browser finds all span elements, traverse up its parent to find p elements).
